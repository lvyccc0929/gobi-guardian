# 戈壁守望者 · GUARDIAN OF THE GOBI## 完整设计方案---## 一、新闻背景2025 年 10 月，一位视频博主在浏览卫星地图时，发现新疆哈密戈壁深处一处异常——1967 年由空军航校学员用石块铺设的巨型大地标语「为人民服务」被越野车碾压破坏，轮胎痕迹清晰可见。这组标语并非孤例。1967 年，空军第八航空学校的学员在哈密戈壁上，用戈壁碎石拼出了五组巨型标语：「为人民服务」「向斗争中学习」「只争朝夕」「排除万难去争取胜利」「毛主席万岁」。每个字占地数十平方米，从万米高空清晰可见——它们当年是飞行学员的地面导航坐标。58 年来，它们扛住了风沙，却没能扛住几脚油门。事件经短视频平台传播后引发广泛关注。博主跨越 2000 多公里从成都前往哈密，跑遍当地矿业公司和砂厂寻找颜色完全匹配的黑褐色砾石，用小型货车分 27 趟转运 20 余吨石料，自发完成修复。2025 年 11 月，哈密市相关部门立案调查，肇事者被找到。同年 12 月，「为人民服务」等五处戈壁标语被纳入哈密市第四次全国文物普查新发现不可移动文物名录。2026 年，哈密市推动戈壁生态保护立法。本作品将这一事件转化为 8 幕 H5 互动叙事体验，用卫星视角、地图穿越、实地触摸、时间流逝等不同镜头语言，让观众从「知道这件事」变成「经历这件事」。---## 二、场景全景```P0 启动 ──→ P1 卫星发现 ──→ P2 奔赴戈壁 ──→ P3 寻石记  (虚空)      (太空 535km)      (地图 2168km)     (戈壁地表)  ──→ P4 戈壁一日 ──→ P5 卫星确认 ──→ 技术：Canvas 2D 粒子动画 + DOM- 时长：约 5 秒自动播放- 无交互### P1 · 卫星发现 ✅**太空视角的发现。** Three.js 3D 地球旋转，程序化生成大陆纹理，双层大气 Shader（内层瑞利散射蓝 + 外层米氏散射白）。用户双指缩放拉近，从 535km 高空逐步下降到地表。接近地面时触发异常检测：终端命令行界面逐行输出异常坐标 `42°27'18"N 93°38'42"E`、异常面积 ≈127m²、置信度 98.7%。确认后进入前后对比模式——拖动滑块对比 2023 年与 2025 年卫星影像，标语区域的轮胎伤痕清晰可见。- 技术：Three.js WebGL + 自定义 Shader + CDN importmap- 交互：双指缩放、点击确认、滑动对比- HUD：坐标显示、高度计、缩放条### P2 · 奔赴戈壁 ✅**地图上的穿越。** 米色做旧地图纹理，从成都到哈密再到戈壁深处的曲线路径。底部道路条拖动控制行程进度（0 → 2168km）。汽车图标从右向左行驶，沿途 11 个地名节点依次弹窗（成都→广元→陇南→天水→兰州→武威→张掖→嘉峪关→瓜州→哈密→戈壁深处），每个节点显示地名和事件。抵达终点后触发无人机视角——沙尘粒子弥漫画面，电影级上下遮幅出现，5 张历史卡片（航校照片、设计图、卫星图、标语实拍）可手动左右滑动翻阅。- 技术：Canvas 2D 地图绘制 + DOM 卡片轮播- 交互：底部拖动 + 左右滑动翻卡片- 历史卡片非自动播放，仅手动翻阅### P3 · 寻石记 ✅**地面上的选择。** 5 层 Canvas 叠加构建戈壁场景：渐变天空 → 4 层山脉剪影（远灰褐→中赭褐→近深褐→前景，山脊散布植被绿斑）→ 戈壁地表（1200+ 颗随机砾石、15 条干裂缝隙、35 簇枯草、6 株梭梭灌木、20 条风蚀沟槽）→ 热浪扭曲（正弦偏移渲染）→ 风沙粒子（120 个低密度飘移 + 迷你旋风每隔 10-18 秒随机生成）。三块漫画风格石头（Canvas 2D 手绘：不规则多边形轮廓 + 粗墨线描边 + 交叉排线阴影 + 矿物脉纹 + 质感点 + 高光线），分别为土黄砾石、米白碱土、黑褐砾石。用户拖拽石头到戈壁地表参考框比对。选择前，金色信息横幅提示：「戈壁地表覆盖着黑褐色砾石，只有找到颜色完全匹配的石料，才能不留痕迹地完成修复。」选错：参考框红色抖动 + 「再仔细看看戈壁地表的颜色」。选对：参考框金色发光 + 金色粒子爆发 + 揭示故事——路遥他们跑遍哈密所有矿业公司和砂厂，终于找到了颜色完全匹配的黑色砾石。- 技术：5 层 Canvas 叠加 + 热浪后处理 + 粒子系统- 交互：拖拽比对- 石块从 6 个 CSS div 堆砌→Canvas 手绘漫画风格### P4 · 戈壁一日 ⏳**时间的陪伴。** 固定机位的戈壁广角镜头，标语「为人民服务」横亘在画面中央。用户横向滑动屏幕控制时间流逝——太阳东升西落，天空色彩从冷蓝→白热→暖橙→深蓝黑循环。极简人影剪影在标语旁循环劳作（弯腰搬运→放置石子→走回→休息）。轮胎伤痕区域的黑色像素逐日被黑褐色砾石替换，每天新增 100-200 颗石子。24 天 = 24 趟运输。底部金色时间线标注 24 个刻度，当前天数高亮。关键天数（4、8、12、16、20、24）的黄昏时刻，画面底部浮现纪录片式字幕，淡入 1s → 停留 3s → 淡出 1s。松手后时间以极慢速度自动流逝（约 1 分钟走完一天）。双击跳到下一天清晨。24 天结束后，画面缓慢后拉，从地面镜头过渡到俯瞰——无缝衔接 P5。- 技术：4 层 Canvas 叠加 + 动态光照 + 人影循环动画- 交互：横向滑动控制时间- 白天风沙粒子 / 夜晚萤火微光粒子，密度随时段过渡### P5 · 卫星确认 ⏳**再回太空，首尾闭环。** 回到与 P1 相同的太空视角。23 秒倒计时——修复后的第 23 天，卫星将再次过顶扫描同一坐标。倒计时数字跳动，卫星扫描线从画面左侧扫至右侧，覆盖标语区域。处理中动画持续 1.5 秒。结果：「异常已消除」——修复前后的卫星对比自动切换，伤痕消失，标语恢复清晰。与 P1 的「检测到异常」形成完整呼应。- 技术：Three.js 复用 P1 场景 + 扫描线 Shader- 交互：等待 + 观看- 关键参数：23 秒倒计时（SATELLITE.COUNTDOWN_SECONDS）### P6 · 回响 ⏳**事件的社会反响。** 深色背景上，真实网络评论像气球一样从底部缓缓浮升，到顶部自动消失。每条评论为半透明圆角气泡，暖色调，有微小的水平漂移。速度偏慢，营造「慢慢看」的节奏。上部叠加可滚动的 6 张时间线卡片（2025.10 卫星发现 → 2025.10 跨越两千公里 → 2025.11 官方立案 → 2025.12 纳入文物普查 → 2026.01 媒体跟进 → 2026 为戈壁立法），最后一张「未来·每一个你」金色高亮。卡片通过向上滑动逐张展现。- 技术：Canvas 2D 评论气球动画 + DOM 时间线- 交互：自动播放评论 + 手动翻阅卡片- 评论素材需从抖音/B站/微博评论区手动采集真实内容### 终章 · 海报 ⏳**传播的起点。** 修复后的「为人民服务」航拍图为主视觉。画面中央大字：「戈壁守望者」。底部：「基于真实事件改编。2025.10 - 2026」。长按保存或分享。从体验者到传播者的转化。- 技术：Canvas 2D 海报合成- 交互：长按保存---## 四、技术架构### 渲染方案（按场景）| 场景 | 渲染方式 | 层数 ||------|----------|------|| P0 |
